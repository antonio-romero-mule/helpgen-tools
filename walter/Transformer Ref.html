Transformer Ref Reference

Purpose:
References a transformer defined as a global element.

Discussion:
A Transformer Ref is a quick way to reference a previously created global Transformer element. You create a global Transformer element from the Global Elements pane. See Configuring Global Mule Elements for instructions on create a global element.

Click the General tab and set the Transformer Reference property to the global transformer name. The global Transformer must have already been created and you select its name from the pull-down list.
