Message Chunk Aggregator Reference

Purpose:
Message Chunk Aggregator combines multiple messages into one message by matching correlation IDs for those messages.

Discussion:
Some outbound routers, such as Splitter routers, set correlation IDs on messages when they dispatch these messages. Messages with a correlation ID can be aggregated back together again using Message Chunk Aggregator.
