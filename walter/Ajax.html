Ajax Reference

Purpose:
The Ajax endpoint component allows Mule events to be sent and received asynchronously to and from a web browser.

Discussion:
In using the Ajax endpoint component, you typically add a JavaScript client to a web page, allowing a web client to listen for events, send events, and perform Remote Procedure Calls (RPC). When you add an AJAX endpoint component to your flow, Mule creates a channel through which Mule events can be asynchronously transferred between your flow and a web page. You define this channel from the General tab of the Ajax property dialog box. The Ajax endpoint component works in conjunction with an Ajax connector. The Connector Ref property, which you define from the References tab, is a reference to the Ajax connector. The Ajax connector is defined outside of your flow and can be created and configured in the Global Elements view.
