Payload Reference

Purpose:
Matches the type of the payload that is contained within the message.

Discussion:
Use a Payload filter to check the class type of the payload object inside a message. You must set the Expected Type property to the class that indicates the expected type of the payload object. For example, if you are expecting a String type, you set Expected Type to java.lang.String.
