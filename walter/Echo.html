Echo Reference

Purpose:
Use Echo to echo or display the message payload.

Discussion:
Other than for an display optional name and description, you can specify an interceptor for an Echo component.

Double-click the Echo icon on the canvas to open the Pattern Properties dialog. Use the Advanced tab to specify an interceptor for Echo.
