FTP Reference

Purpose:
File Transfer Protocol (FTP) Endpoint carries message data to and from a remote server.

Discussion:
Placing an FTP Endpoint in your flow integrates your message data with an FTP server. When an FTP Endpoint is used to receive message data (when it is an Inbound FTP Endpoint), Mule ESB monitors the FTP server for new data at the polling frequency you specify. When new data arrives on the server, this endpoint delivers it to the flow. Likewise, when used to send message data (when it is an Outbound FTP Endpoint), once message data arrives at the endpoint Mule ESB logs in and writes data to the specified FTP server.
