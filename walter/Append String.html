Append String Reference

Purpose:
Appends a string to a string payload.

Discussion:
The Append String Transformer has the same advanced properties as all other transformers. See http://www.mulesoft.org/documentation/display/MULE3STUDIO/Configuring+Transformers+in+Mule+Studio to configure these optional properties.

You must specify the string that the transformer appends to the message payload. Click the General tab and place the string in the Message field.

Use the Advanced tab to optionally configure the properties standard to all transformers.
